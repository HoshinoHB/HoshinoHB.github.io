## 🚀 部署&访问
本项目已经全部配置完成：
1. 进入仓库 `Settings → Pages`
2. 部署来源选择 `Deploy from a branch`
3. 分支选 `main`，目录选择 `/ (root)`
4. 等待2分钟自动构建完成

✅ 永久访问地址：
`https://hoshinohb.github.io/`

## 🛠️ 日常维护教程
1. **新增粉丝头像**
打开`index.html`，在对应平台的粉丝数组里，追加「头像图床链接+个人主页地址」即可。

2. **修改页面外观**
直接修改`<style>`标签内的颜色、渐变、尺寸参数，一键更换整体风格。

3. **评论后台管理**
所有留言、点赞、回复，全部同步保存在仓库`Discussions`板块，可随时审核/删除/回复。
